## My Movies:
### About:

*My Movies* is a simple, responsive website generated using python. The site features information about some of my favorite movies with infromation pulled from [The Open Movie Database](https://www.omdbapi.com/) and [youtube](www.youtube.com).

This project was created in partial fulfillment of Udacity's FullStack Nanodegree program, and is based on a starter file created by Udacity.

You can find the original file here:
https://github.com/adarsh0806/ud036_StarterCode/blob/master/fresh_tomatoes.py

### Requirements:

+ python >= 2.7
+ a modern web browser with Javascript enabled

### Run:
The easiest way to run this program is via the command line.

  1. Fork or download this repository.
  2. Navigate to your download directory (uncompress if necessary).
  3. Use the python command to run entertainment_center.py:
 
 `$ python entertainment_center.py`
